# 更新日志

格式参考：每个版本用 `## vX.Y.Z (YYYY-MM-DD)` 标题，下方用 Markdown 列表描述更新内容。

## v1.0.8 (2026-07-09)

### 新增功能

- 新增自定义网络壁纸API（用户可填写自己的壁纸API，不支持分类）
- 新增本地视频来源（Beta）：支持选择本地视频作为屏保循环播放
- 新增URL视频来源（Beta）：支持网络视频URL循环播放
- 新增播放视频声音开关（仅视频源时显示）
- 新增娱乐模式一键开关：设置页面可一键开启/关闭娱乐模式
- 新增开机自启动配置教程（注册表方式）

### 优化改进

- 优化设置窗口：固定大小不可调整，避免误操作
- 优化滚动条：滚动条从标题栏下方开始显示，不再遮挡标题栏
- 优化娱乐模式：UI 和托盘菜单状态实时同步
- 优化跨平台标题栏：Windows 自定义按钮 / macOS 红绿灯 / Linux GNOME 风格按钮
- 优化 macOS 适配：自动创建系统菜单栏（应用菜单、编辑菜单、窗口菜单）

### 调试工具

- 新增 `--win` / `--macos` / `--linux` 命令行参数，用于在当前系统下模拟其他平台标题栏样式

### 协助测试

- 由于开发者没有 MacBook 设备，**macOS 平台的系统菜单栏无法完整测试**，欢迎 Mac 用户下载源码并编译后在「关于 / 编辑 / 窗口」菜单中验证功能是否正常
- 测试方法：设置页面底部"更新日志"按钮 → 查看 macOS 红绿灯和系统菜单栏效果

### 系统支持说明

- **由于技术原因，1.0.8 暂时只能提供 Windows 安装包**（NSIS 安装包 + 便携版）
- macOS / Linux 平台的代码已就绪（跨平台标题栏、macOS 系统菜单栏等），但**由于打包/签名工具链和开发设备的限制**，暂无法直接发布对应平台的安装包
- 后续版本会逐步补齐 macOS 和 Linux 的安装包，敬请期待

## v1.0.0 (2026-07-04)

首款正式版本发布。

### 功能特性

- 多种壁纸来源：网络壁纸、自定义URL、多张URL、本地单图、本地文件夹
- 自动切换壁纸，可自定义切换间隔
- 两种退出方式：仅手动关闭 / 任意动作退出
- 实时时钟显示，可选位置和字体
- 视觉特效：淡入淡出过渡效果
- 娱乐模式：暂停屏保计时
- 预览窗口：实时预览屏保效果
- 深色主题界面
